### Class: X2Editbox

> [X2Editbox](../types/X2Editbox.md#class-x2editbox)
> 
> A `X2Editbox` widget is an **working** extension of the `Editbox` widget,
> providing a text input field for user interaction. It inherits all
> functionality from `Editbox`.
